**WPF-приложение** для автоматизации работы медицинского учреждения, разработанное с использованием паттерна **MVVM**.

### Основные возможности

* **Авторизация и роли**: Разграничение доступа для различных типов пользователей (например, врачей и регистраторов).
* **Управление пациентами**: Добавление новых пациентов, редактирование их данных и ведение истории приемов.
* **Кабинет врача**: Интерфейс для просмотра записей, работы с медицинскими картами и назначениями.
* **Регистратура**: Функционал для оформления записей на прием и первичной обработки данных.
* **Валидация**: Проверка вводимых данных (номера телефонов, ID, текстовые поля) с использованием кастомных правил валидации.
* **Динамические темы**: Поддержка темного и светлого оформления интерфейса.

### Технологический стек

* **Backend**: .NET 9.0 (WPF).
* **Хранение данных**: Использование JSON-файлов для локального хранения информации о врачах, пациентах и историях приемов.
* **UI/UX**: Material Design In XAML Toolkit для современного визуального стиля.
* **Логика**: Паттерн RelayCommand для обработки действий пользователя.

### Структура проекта

* **Models**: Сущности `Doctor`, `Pacient`, `AppointmentStories`.
* **ViewModels**: Логика взаимодействия между данными и интерфейсом (`MainViewModel`, `RegPageViewModel` и др.).
* **Views**: XAML-описания страниц и окон приложения.
* **ValidationRules**: Правила для проверки корректности заполнения форм.
